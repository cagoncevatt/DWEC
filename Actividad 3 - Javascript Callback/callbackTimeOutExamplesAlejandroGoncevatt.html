Conslusiones y Reflexiones

El uso de las funciones en Javascript, a primera vista, no parece cambiar tanto de la forma de uso de otros lenguajes de programación. El primer gran cambio es el poder pasar funciones como tal como parámetros a otras funciones de manera que puedan ser usadas para poder definir un orden de ejecución variante y dinámico, aunque aún atados en cierta manera a la ejecución secuencial.

Por otro lado Javascript nos da funciones para poder simular la asincronía, esto es a través de una función a la que puedes enviar una función (declarada allí mismo o ya declarada en otro punto del código) y definir si quieres que se ejecute de manera constante cada X tiempo, o si quieres que esa misma función se ejecute luego de X tiempo de que se "prepara".

En definitiva si se ven las posibilidades y se aprovechan bien estos recursos que nos ofrece Javascript se pueden lograr algunas secuencias de funciones muy útiles y que pueden dotar nuestra web de un aspecto o características únicas respecto a la muestra o gestión de la información.
